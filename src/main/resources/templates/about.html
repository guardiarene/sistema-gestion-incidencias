Acerca del Proyecto

Este proyecto es un Sistema de Gestión de Incidencias, diseñado para facilitar la administración y el seguimiento de las consultas, sugerencias, reclamos, peticiones y quejas reportadas por los usuarios.

Tecnologías Utilizadas

  • Java
  • Spring Boot
  • MySQL
  • Thymeleaf
  • HTML5
  • CSS3
  • Bootstrap
